Py-Attire-Logger

ATTiRe stands for attack tool timing and reporting. This is a machine-readable JSON formatted superset of useful attack simulation reporting data for content creation and enrichment in tracking tools like VECTR. It uses a structured log format to ease the creation of reporting detail and categorisation of red team activities. Several commercial and open-source red team operator tools use proprietary logging formats to record the results of adversary emulation and attack simulation. Normalization and translation of this log data can be challenging. At the moment these ATTiRe log files are being manually created.

The goal for this Attire Logger project is to develop a program that will dramatically increase the efficiency at which this information is collected and then formatted into the correct layout before it is then subsequently uploaded to Vectr. Once the program has been tested extensively the goal is to roll it out to the cyber security professional community.
